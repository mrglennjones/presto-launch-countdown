# presto-launch-countdown
T- Countdown for the next space flight rocket launch on a PImoroni Presto with added nebula ambient lighting and pulsing red led 30 minute warning, so youll never miss a launch again!

powered by Launch Library 2 from thespacedevs.com

### guide
+ enter you WIFI_SSID and WIFI_PASSWORD to secrets.py
+ create gallery/ on your SD card
+ run main.py
